初心所系

项目缘起

聚焦传统文化技艺的活态传承，以数字化手段记录匠人与手艺的生命轨迹。

目标人群

聚焦非遗传承人、乡土匠人及其家庭，携手文化机构共建数字化记忆库。

在时代快速变迁的浪潮中，许多技艺与讲述正悄然隐没。我们希望通过“记艺拾光计划”，为这些技艺守住声音、留下记忆。 项目以AI智能体与非遗传承人之间的对话为核心采集方式，引导他们自然讲述手艺与人生故事，经过结构化整理后，生成可共享、可归档、可传承的非遗数字档案。 工程不仅关注“技艺如何做”，更注重“人如何传”，希望为手艺背后的个体生命经验留痕，为传统文化的数字化转译提供温柔路径。我们特别关注那些处于边缘化的乡土工艺、老匠人族群，尤其是手艺传承即将中断的“断层地带”，通过系统采集与生成机制，推动非遗保护从“静态展示”向“动态留声”迈进。

智忆传承

项目介绍

聚焦非遗技艺与人生记忆的数字化保存，借助AI技术构建面向传承者的文化档案平台。

核心价值

用科技赋能非遗口述采集，让每位传承人的故事被听见、被整理、被传承。

“记艺拾光工程”是专为非遗传承人设计的AI口述采集与档案生成系统，融合语言识别、情感交互、结构化抽取等语言科技核心模块，基于「对话式引导+智能化处理」，构建面向全国非遗口述内容的数字档案库，让每一门技艺、每一段人生都能被记录、看见、传承。

  • 【拾艺典藏】（非遗数字库建设）： 聚焦国家级、省市级非物质文化遗产项目及乡土手艺，采集技艺流派、代表性传承人、作品工序与传承路径，构建结构化数字档案，实现“人-艺-史”联结可查可续。
  • 【乡语留声】（方言可识别）： 系统支持多种汉语方言（如晋语、吴语、赣语、闽南语等）语音采集与智能转写，贴近地方非遗表达语境，打破普通话采集壁垒，增强文化真实感。
  • 【技艺成文】（一站式档案服务）： 语音-文字-档案一键生成，内容结构自动适配Word文档/年表/代表作品图文档，支持打印装帧与网页分享，打造“说出来就是一份档案”的非遗讲述体验。
  • 【文化映照】（文化价值延伸）： 系统自动关联技艺所属历史节点（如清代起源、战后复兴等），补足非遗“讲得清”的断裂，为非遗讲述注入文化背景脉络与精神价值。
  • 对传承人
    • 打破“写不出来、说不清”的记录困境，只需自然对话就能生成完整档案
    • 让个人讲述不再散落，通过声音和语言将技艺人生升格为可传承的文化文本
    • 赋予“我讲的故事就是历史”的主体尊严与自我认同
  • 对普通人（非从业者）
    • 拉近大众与非遗的距离，听懂“身边的手艺”“邻里的故事”
    • 感受一代人用双手留下的纹理，形成“技艺即生活”的文化认知
  • 对学习者 / 研究者
    • 提供真实语料与口述档案，为文化学、人类学、语言学等提供新型素材
    • 非遗传承与语言演变并置，支持“方言+技艺”双域研究
    • 可与知识图谱/语义网络结合，构建AI驱动下的文化链式学习系统
  • 对社会
    • 以“活的技艺”形成动态文化底图，提升公众文化共鸣感与保护意识
    • 打破精英记录垄断，让每一位传承人都能拥有自己的“文化身份证”
    • 探索“文化共建+科技共创”的非遗活化新路径
