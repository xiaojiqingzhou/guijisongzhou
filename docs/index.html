初心所系

项目缘起

聚焦非遗传承保护难题，以科技之力构建匠艺守护平台

目标人群

聚焦非遗传承人与民间艺人，连接家庭传承者与文化守望者共建数字技艺档案

在与各地非遗传承人深度访谈中，我们触摸到时间沉淀的匠心与风骨。从景泰蓝烧制、剪纸刻刀到木雕传神，数百项传统技艺在传承链条中正面临断裂。我们深耕这一困境，打造AI驱动的“技艺口述史”数字化工具，让技艺流转不再仅靠师徒，而可数字永存。

  • 核心主角：国家/省级非遗传承人、地方工艺艺人、传统匠人
  • 同行伙伴：
    • 关注家族技艺的「传承家庭」
    • 推动文化采录的「地方机构」（非遗中心/文化馆/高校）
    • 记录文化脉络的「历史采录者」（学者/研究者/志愿者）
  • 特别场景：
    • 传统节日里的 “工艺重现”
    • 展览馆、文旅市集的 “数字展陈”
    • 课堂教学与传承课程的 “非遗素材库”

智忆传承

项目介绍

关注非遗口述困境，以AI之力构建“技艺+故事”双轨数字档案体系

核心价值

通过语音+图像+文字融合采录，让非遗不止被看见，更被理解与传承

系统融合语音识别、情感识别与图像生成等技术，围绕非遗传承人讲述，形成沉浸式数字化非遗档案：

  • 方言讲述适配：AI数字人支持粤语、吴语等方言输入并精准识别，打破传统访谈的隔阂感
  • 工艺图像补全：结合讲述内容自动生成插图与技艺工序示意图
  • 结构化生成：输出图文并茂的非遗档案，包括简介、流程、背后故事
  • 对传承人：记录技艺心路历程，打造专属数字作品集
  • 对机构：支持博物馆/高校/文旅平台进行标准化非遗素材采集
  • 对社会：形成全民可阅读、可传播的技艺知识图谱
